# CRM システム

Ruby on rails ベースの包括的な顧客関係管理 (CRM) システムです。

## 概要

このプロジェクトは、企業の顧客管理、製品管理、店舗管理、店員管理を統合的に行うためのWebアプリケーションです。

## 主要機能

### 認証機能
- ユーザーログイン/ログアウト
- セッション管理
- 認証必須ページ保護

### 顧客管理
- 顧客登録・編集・削除
- 顧客一覧表示
- 顧客詳細情報表示

### 製品管理
- 製品登録・編集・削除
- 製品一覧表示
- 製品詳細情報表示

### 店舗管理
- 店舗登録・編集・削除
- 店舗一覧表示
- 店舗詳細情報表示

### 店員管理
- 店員登録・編集・削除
- 店員一覧表示
- 店員詳細情報表示

## 技術仕様

- **フレームワーク**: ruby on rails 8
- **データベース**: MySQL
- **開発サーバーポート**: 8001

## データベース設定

```
データベース: mysql
ホスト: localhost
ポート: 3306
データベース名: crm2
ユーザー名: crm2user
パスワード: 1234
```
